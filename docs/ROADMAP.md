# Roadmap

## Next Engineering Milestones

1. Add OAuth providers for Gmail, Outlook, Google Calendar, and Microsoft Calendar.
2. Add an approval queue endpoint and dashboard approval controls.
3. Replace deterministic local semantic vectors with selectable OpenAI, Gemini, or local embedding providers.
4. Add Whisper and Deepgram streaming STT adapters.
5. Add Piper process integration for offline TTS.
6. Expand the video agent into a queue-backed FFmpeg, OpenCV, and Whisper pipeline.
7. Add LangGraph state machine persistence for long-running tasks.
8. Add smart home adapters through Home Assistant MCP tools.
9. Add browser session memory and authenticated website workflows.
10. Add plugin-style tool discovery from local MCP servers.
